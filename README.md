# Analyse-log

Analyse-log est un outil d’analyse de fichiers de logs au format CSV, réalisé dans le cadre d’une PFMP en entreprise.

## Fonctionnalités

Cet outil permet de :
- Trier les logs par adresse IP source
- Trier les adresses IP distinctes par domaine
- Trier les adresses IP distinctes par catégorie
- Visualiser le pic d’activité

## Options d’analyse

L’outil intègre plusieurs options afin de simplifier l’analyse :
- Recherche par date précise
- Recherche par heure et/ou minute précise
- Affichage du top N des adresses IP sources les plus actives
- Affichage du top N des adresses IP distinctes par domaine
- Affichage du top N des adresses IP distinctes par catégorie
